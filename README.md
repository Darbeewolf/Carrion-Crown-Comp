# Carrion Crown Compendium Unofficial
Compendium Pack of Actors, Items, Rolltables, & Conversion Journals for Books 1-3,5-6 of Carrion Crown. Maps included are by commmunity contributions.
This covers level 1 to 13 play.

**To Manually install this module:**
1. Click Install Module or Install by Manifest on Forge
2. Paste the following into the Manifest URL: https://github.com/Darbeewolf/Carrion-Crown-Comp/releases/latest/download/module.json
3. Click Install module.

Future Expansion: Expand on Maps, Add journal placeholder for Book 4.
